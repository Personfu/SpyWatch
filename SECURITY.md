# Security Policy

SpyWatch is a defensive research repository.

## Report content issues

Open an issue or pull request if you find:

- unsupported attribution;
- stale source links;
- privacy-sensitive material;
- harmful procedural detail;
- duplicate actor identities;
- incorrect alias mapping.

## Do not submit

Do not submit private records, private contact data, passwords, access tokens, intrusion instructions, or content that enables harm.
