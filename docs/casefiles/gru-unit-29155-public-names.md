# Casefile: GRU Unit 29155 public names and source handling

## Purpose

This casefile records public names and source handling notes for GRU Unit 29155. It is defensive research material only. It does not include private addresses, private relatives, copied private images, or instructions for finding people.

## Public-source anchors

- Rewards for Justice: GRU Officers - Unit 29155

## Summary

Rewards for Justice publicly identifies several Russian military intelligence officers with Unit 29155 of the Russian General Staff Main Intelligence Directorate. The page connects the public names to malicious cyber activity involving Ukrainian government systems and critical infrastructure, including WhisperGate malware, and says activity also targeted U.S.-based critical infrastructure sectors.

## SpyWatch handling rules

1. Treat Rewards for Justice and DOJ pages as official source anchors.
2. Treat vendor cluster names as aliases that may not map perfectly to individuals.
3. Do not add personal information beyond official public case context.
4. Link to official images or posters instead of copying them into the repo.
5. Separate cyber-activity reporting from physical-world hybrid-operation reporting.

## Defensive lessons

1. Public advisories can connect malware, infrastructure targeting, state sponsorship, and named public figures, but defenders should still avoid overbroad attribution claims.
2. Critical infrastructure organizations should focus on hardening, identity controls, backup resilience, logging, and vendor-access review.
3. Actor profiles should cite exactly which source supports each name or alias.
