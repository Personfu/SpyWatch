## Public action connections

- Iran-linked clusters are often tracked by mission, infrastructure, and victimology rather than a named public unit.
- SpyWatch should map aliases for research, but it should not invent hard unit claims without official public support.
- Some sources associate MuddyWater with Iranian government security structures, but this page keeps the unit field conservative unless a casefile cites a specific public source.

## Handling rule

Keep alias normalization separate from unit attribution. A vendor name is a cluster label, not a court finding.
