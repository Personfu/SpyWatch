# Defender Briefing Templates

Use these templates to convert actor profiles into practical, non-operational briefings.

## One-page actor brief

```text
Actor:
Aliases:
Country / alignment:
Confidence:
Primary public sources:
Sectors mentioned in reporting:
Why leadership should care:
Most important defensive lessons:
Open questions:
Last reviewed:
```

## Executive summary format

```text
This actor matters because public reporting links it to [sector/context]. The main defensive lesson is [plain-language risk]. The organization should prioritize [controls] and should avoid overreacting to unverified claims.
```

## Student research format

```text
1. Read three reliable sources.
2. Identify alias differences.
3. Separate facts from analysis.
4. Write a neutral summary.
5. List defensive lessons.
6. Explain what is still unknown.
```
